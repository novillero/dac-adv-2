# Инструкция по изменению имени пользователя в Telegram (Десктоп версия)
1. Откройте приложение [Telegram](https://desktop.telegram.org/?setln=ru).

2. Войдите в аккаунт.

3. Нажмите на кнопку **Бургер** в левой панели управления.

    ![скриншот1](png/1.png)

4. Выберите пункт меню Мой профиль.

    ![скриншот2](png/2.png)

5. Нажмите на кнопку **Карандаш** в правом верхнем углу.

    ![скриншот3](png/3.png)

6. Выберите пункт Имя пользователя.

    ![скриншот4](png/4.png)

7. Измените имя пользователя согласно требованиям.

    ![скриншот5](png/5.png)

8. Сохраните изменения.
