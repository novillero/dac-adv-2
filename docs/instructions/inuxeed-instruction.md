# Как в Телеграм поделиться постом в истории.
## Для того, чтобы поделиться постом в истории необходимо выполнить следующие действия:
1. Войти в свой Телеграмм-аккаунт;
2. Открыть интересующий телеграм-канал;
2. Выбрать в телеграм-канале интересующий пост;
3. Нажать на стрелку в нижнем правом углу поста, при этом откроется впадающее меню;
4. Выбрать "Репост в историю", при этом начнется загрузка истории;
5. Убедиться, что история загрузилась.
