# Как в Телеграм поделиться постом в истории

Для того, чтобы поделиться постом в истории необходимо выполнить следующие действия:

1. Войдите в свой телеграм-аккаунт.

2. Откройте интересующий телеграм-канал.

2. Выберите в телеграм-канале интересующий пост.

3. Нажмите на стрелку в нижнем правом углу поста, при этом откроется выпадающее меню.

4. Выберите **Репост в историю**, при этом начнется загрузка истории.

5. Убедитесь, что история загрузилась.
