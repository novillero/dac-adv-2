# Как в Телеграм поделиться постом в истории

Для того, чтобы поделиться постом в истории выполните следующие действия:

1. Войдите в свой телеграм-аккаунт.

2. Откройте интересующий телеграм-канал.

3. Выберите в телеграм-канале интересующий пост.

4. Нажмите на стрелку в нижнем правом углу поста, при этом откроется выпадающее меню.

5. Выберите **Репост в историю**, при этом начнется загрузка истории.

6. Убедитесь, что история загрузилась.
