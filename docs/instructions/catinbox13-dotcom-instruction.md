# Экспорт  материалов из Telegramm на персональный компьютер
---

Приложение Telegramm даёт возможность экспортировать материалы (такие как фотографии, видео, голосовые и видео сообщения, стикеры, файлы или GIF), которыми вы обмениваетесь в чате, в папку на ваш персональный компьютер.

Для того, чтобы экспортировать материалы:

1. Откройте чат с нужным пользователем.

2. В правом верхнем углу нажмите на значок с тремя вертикальными точками (см. рисунок 1).

![рис. 1]/(images/cnb_img/three_dots.png)

3. В выпадающем списке выберите *Export chat history*, как показано на рисунке 2.

![рис. 2](/images/cnb_img/list.png)

4. В открывшемся окне *Chart export settings* отметьте чекбоксы для одного или нескольких варианов, см. рисунок 3:

    * *Photos* - фотографии;

    * *Videos* - видео;

    * *Voice messages* - голосовые сообщения;

    * *Video messages* - видео сообщения;

    * *Stickers* - стикеры;

    * *GIFs* - файлы в разрешении `.gif`;

    * *Files* - файлы.

    ![рис. 3](/images/cnb_img/exp_sttngs.png)

5. Перетащите бегунок регулировки максимального размера экспортируемых файлов.

6. Выберите формат экспортирования:

    * для параметра *Format* нажмите на текущий вариант формата (например `HTML`), см. рисунок 4;

    * в открывшемся окне отметьте одну из радиокнопок, см. рисунок 4:

        - *Human-readable HTML* - человекочитаемый `HTML`;

        - *Machine-readable JSON* - машиночитаемый `JSON`;

        - *Both* - оба формата.

        ![рис. 4](/images/cnb_img/format.png)


    * нажмите на кнопку **Save**;

    * для параметра *Path* нажмите на текущий вариант пути, куда будут сохранены дланные;   

    * в открывшемся окне выберите нужный путь и нажмите на кнопку **Выбор папки**, см. рисунок 5;

    ![рис. 5](/images/cnb_img/path.png)

    * для параметра *From* нажмите на текущий вариант начальной нужжной точки с которой будет произведён экспорт  (например *the oldest message*);

    * в открывшемся окне с календарём выберите нужную дату и нажмите на кнопку **Reset**, см. рисунок 6;
    
    ![рис. 6](/images/cnb_img/calendar.png)

    * для параметра *to* нажмите на текущий вариант конечной нужной точки с которой будет произведён экспорт  (например *зкуыуте*);

    * в открывшемся окне с календарём выберите нужную дату и нажмите на кнопку **Reset**;

7. В окне *Chart export settings* нажмите на кнопку **Export**.

8. В открывшемся окне *Export Your Data* отобразится информация о загрузке данных, как на рисунке 7:

    ![рис. 7](/images/cnb_img/exp_y_data.png)

    * *Data export ...* - статус загрузки данных;

    * *Total files* - количество загруженных файлов;

    * *Total size* - размер экспортируемых файлов.

>После завершения экспортирования данных появится запись "Your data was sucsessfully exported.".

9. Окно можно закрыть, нажав на иконку "крестик" или открыть папку с экспортированными файлами, для этого нажмите на кнопку **Show my data**.
