# Как привязать почту в мессенджере Telegram
Привязка адреса почты позволяет авторизоваться по ней в Telegram, если получить код авторизации по телефону не удается. Чтобы привязать почту:
1. Откройте приложение Telegram.
2. Откройте раздел **Настройки** > **Конфиденциальность** 
3. В открывшемся списке настроек выберите **Почта для входа**. По умолчанию эта настройка отключена.
4. Выпадет окошко с предупреждением, что на этот адрес почты будут приходить коды авторизации. Нажмите **ОК**.
5. В открывшемся окне введите свой адрес почты и нажмите **Подтвердить**. Откроется окно ввода кода.
6. Откройте почту, которую вы указали в предыдущем окне, и введите полученный на нее код в окошко.

Готово: почта привязана к вашему аккаунту Telegram и отображается в списке настроек конфиденциальности.
