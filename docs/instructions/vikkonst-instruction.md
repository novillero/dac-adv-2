# Как создать папку с чатами в Telegram: инструкция для iOS

Папки помогают группировать чаты по темам. Например, можно создать отдельные папки для личных сообщений, работы или учёбы.

Чтобы создать папку с чатами:

1. Откройте приложение Telegram.

2. Нажмите кнопку **Настройки** (иконка шестерёнки в правом нижнем углу).

3. Выберите пункт **Папки с чатами**.

4. Нажмите кнопку **Создать папку**.

5. Введите название папки в поле «Название папки».

6. Нажмите кнопку **Добавить чаты**.

7. Выберите нужные чаты и нажмите кнопку **Готово**.

8. Нажмите кнопку **Создать**.

Папка появится как отдельная вкладка в верхней части списка чатов.
