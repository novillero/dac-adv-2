#Как выложить историю в Телеграм

## Шаг 1. Откройте приложение Телеграм 

Убедитесь, что у вас установлена последняя версия приложения - проверьте обновления в App Store или Google Play.

## Шаг 2. На главном экране с чатами нажмите кнопку **История** в верхнем правом углу. 

## Шаг 3. Нажмите кнопку «+» или иконку **Добавить историю**.
gi
## Шаг 4. Выберите источник контента: 

* **Камера** для создания фото/видео;

* **Галерея** для загрузки существующих файлов;

* **Текст** для создания текстовой истории.

Для фото/видео: сделайте снимок или запишите видео длительностью до 60 секунд.
Для выбора из галереи: выберите нужный файл формата: JPG, PNG, MP4 или MOV.

## Шаг 5. Отредактируйте контент: добавьте текст, стикеры, эмодзи, фильтры, нарисуйте что-то от руки.

## Шаг 6. Настройте параметры видимости: выберите, кто может видеть вашу историю (все, только контакты, выбранные контакты, исключенные контакты).

## Шаг 7. Нажмите **Опубликовать** или стрелку в правом нижнем углу.

История опубликуется на сутки с момента размещения.
