# Как отправить отложенное письмо в Telegram
Для отправки отложенного сообщения выполните следующие шаги:
1. Найдите чат/канал, КУДА нужно отправить сообщение
2. Напишите сообщение
3. Зажмите кнопку **«Отправить»** или кликните ПКМ на кнопке **«Отправить»** (доступный вариант зависит от Вашего устройства)
4. В открывшемся меню нажмите на пункт **«Запланировать отправку»** — откроется окно для ввода параметров времени и даты
5. Введите дату и время отложенной отправки сообщения. 
***Примечание***:*если у Вас установлена версия Премиум, то, помимо даты и времени будет доступно поле* ***«Повтор»*** для отложенной отправки регулярных сообщений
6. Если передумали, нажмите кнопку  **«Отмена»**
7. Подтвердите введенные данные кпопкой «Запланировать». После нажатия откроется канал с отражением всех запланированных Вами сообщений.
