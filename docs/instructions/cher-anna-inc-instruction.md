# Очистка кэша приложения Telegram

В случае возникновения проблемы переполнения памяти мобильного устройства, обратите внимание на кэш мобильного приложения Telegram. Очистка данного кэша позволит освободить значительное количество памяти мобильного устройства.

Очистка кэша может производиться как в ручном режиме, так и автоматически.

## Очистка кэша приложения Telegram в ручном режиме

Для выполнения экспресс-очистки кэша приложения Telegram выполните следующие действия:

1. Войдите в приложение Telegram.

2. Перейдите в настройки приложения, выбрав пункт **Settings** основного меню снизу.

3. В открывшемся окне выберите пункт меню **Данные и память**.

4. В открывшемся окне **Data and Storage** выберите пункт меню **Storage Usage** и нажмите на кнопку **Очистить кэш**.

Этот способ мгновенно освобождает до 30% пространства, занятого приложением Telegram.

## Настройка автоматической очистки кэша в приложении Telegram

Чтобы не производить очистку кэша приложения Telegram в ручном режиме каждый раз, когда память мобильного устройства переполняется, можно включить функцию автоматической очистки. Для этого:

1. Войдите в приложение Telegram.

2. Перейдите в настройки приложения, выбрав пункт **Settings** основного меню снизу.

3. В открывшемся окне выберите пункт меню **Данные и память**.

4. В открывшемся окне **Data and Storage** выберите пункт меню **Storage Usage**.

5. В открывшемся окне **Storage Usage** в блоке **Автоудаление закэшированных медиа** установите период времени, по прошествии которого в личных чатах, группах, каналах и историях будет автоматически удаляться кэш. Минимальный период времени -- один день.

6. В поле **Максимальный размер** кэша установите ползунок на минимальное значение (пять гигабайтов).

7. Вернитесь на главную страницу приложения.

Теперь очистка кэша в приложении Telegram будет производиться автоматически.
