# Как создать канал в Telegram

*Примечание.* Инструкция для смартфонов на ОС Android.

Чтобы создать канал в Telegram:

1. Откройте приложение.

3. Опционально. Если у вас несколько аккаунтов, выберите нужный.

5. Перейдите в раздел **Профиль** → **Настройки**.

7. Нажмите **Аккаунт**.

9. Прокрутите экран вниз.

11. Нажмите **Добавить личный канал**.

13. Нажмите **Создать канал**.

15. Заполните поля:

    - «Название канала»
    
    - «Описание канала»
    
17. В правом верхнем углу нажмите галочку.

19. Выберите тип канала:

    - **Публичный** — придумайте уникальную публичную ссылку.
    
    - **Частный** — скопируйте ссылку-приглашение. Вы сможете сделать это сейчас или позже.
    
21. В правом верхнем углу нажмите галочку. 

Теперь у вас есть канал, в котором можно писать что-нибудь интересное, приглашать подписчиков из числа своих контактов и делиться ссылкой-приглашением.
