# Как создать канал в Телеграм

*Примечание.* Инструкция для смартфонов на ОС Android.

Чтобы создать канал в Телеграм:
1. Откройте приложение.
2. Опционально. Если у вас несколько аккаунтов, выберите нужный.
3. Перейдите в раздел **Профиль** → **Настройки**.
4. Нажмите **Аккаунт**.
5. Прокрутите экран вниз.
6. Нажмите **Добавить личный канал**.
7. Нажмите **Создать канал**.
8. Заполните поля:
    - «Название канала»
    - «Описание канала»
9. В правом верхнем углу нажмите галочку.
10. Выберите тип канала:
    - **Публичный** — придумайте уникальную публичную ссылку.
    - **Частный** — скопируйте ссылку-приглашение. Вы сможете сделать это сейчас или позже.
11. В правом верхнем углу нажмите галочку. 

Теперь у вас есть канал, в котором можно писать что-нибудь интересное, приглашать подписчиков из числа своих контактов и делиться ссылкой-приглашением.
