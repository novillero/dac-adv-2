# Как сменить аватар в Telegram на мобильном устройстве

1. **Откройте Telegram** — запустите приложение на телефоне.
2. **Перейдите в меню профиля** — нажмите на икону "Профиль" в нижнем меню страницы.
3. **Нажмите «Выбрать фото»**  нажмите на кнопку изменения фото.
4. **Выберите источник** — сделайте новое фото или выберите изображение из галереи.
5. **Сохраните изменения** — обрежьте фото при необходимости и подтвердите выбор.
