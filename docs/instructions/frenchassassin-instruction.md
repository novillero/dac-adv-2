# Создание приватного канала

Это пошаговая инструкция о функциональности Telegram

1. Откройте приложение Telegram.

2. Нажмите кнопку **Написать сообщение** в правом верхнем углу.

3. Выберите пункт **Создать канал**.

4. На странице «Что такое канал?» нажмите кнопку **Создать канал**.

5. Введите название и при необходимости описание канала, затем подтвердите создание.

6. На шаге выбора типа канала выберите пункт Частная ссылка.

7. При необходимости скопируйте ссылку-приглашение — по ней пользователи смогут подписаться на канал.

8. Добавьте первых подписчиков из списка контактов или пропустите этот шаг.

9. Нажмите кнопку **Далее**, чтобы завершить создание канала. Приватный канал готов к использованию.
