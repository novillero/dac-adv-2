# Настройка папки с чатами

1. Создание папки с чатами:

    - Откройте настройки Telegram.

    - Перейдите в раздел "Папка с чатами".

    - Нажмите кнопку **Создать новую папку**.

    - Придумайте название (например, "Друзья", "Работа" или "Учеба").

2. Добавление чатов:

     - В блоке "Выбранные чаты" нажмите кнопку **Добавить чаты**.

     - Сохраните настройки папки.
  
!!! warning "Важно"
    В одну папку можно добавить до 100 чатов - для обычных пользователей и до 200 - для подписчиков Telegram Premium.

3. Использование:

    - Папки располагаются в верхней части экрана под строкой поиска (рядом с разделом «Все чаты»).

    - Нажмите на нужную папку, чтобы увидеть её содержимое.
