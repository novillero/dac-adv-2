# Как изменить фото профиля в приложении Telegram для ПК
Для изменения фото профиля в приложении Telegram для ПК (персонального компьютера):
1. Кликните по бургер-меню (три горизонтальные полоски) в верхнем левом углу.
2. Нажмите на поле "Мой профиль".
 
3. Далее нажмите на значок "Карандаш" в верхнем правом углу. 
4. Нажмите на значок "Камера" рядом с фото профиля.
  
5. Выберите, как вы будете загружать фото:
 * файл - добавите любое изображение из ПК
 * камера - сделаете фото с внешней камеры вашего ПК
 * выбрать эмодзи - вы можете выбрать любой стикер или эмодзи, которые есть в ваших сохранённых наборах.
6. Например, вы выбрали "Файл": 
 * найдите в вашем компьютере фотографию, которую вы хотите добавить в ваш профиль
* выделите область для отображения
* нажмите "Установить фотографию"
7. Фото профиля обновлено.
